About Project

This is what we did.

For the sentimental Analysis of Hindi Text ,firstly we required a sufficiently large dataset of reviews for Hindi Text ,as no dataset is available already for Hindi Text so we had to scrapped it from various websites. These are some of the websites:-

1. Jagran.com
2. NavBharat Times.com
3. Filmibeat.com
4. Aajtak.com

The purpose of this project is to a create database for sentiment analysis of Hindi text. There exists a wide variety of databases of English text for sentiment analysis such as Amazon product review database, however datasets available for Hindi language are very limited or not up to the mark for some specific purposes. In this project we aim to create one of the largest Hindi text database for sentiment analysis and natural language processing

These are some of the technology used:

    ML: Machine Learning
    NLP: Natural Language Processing
    JSON: Javascript Object Notation.
    >CSV: Comma separated value
    ULMFIT: Universal Language Model Fine-Tuning
    iNLTK: Natural Language Toolkit for Indic Languages
    CLTK: Classical Language Toolki
